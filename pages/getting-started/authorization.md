# Authorization

We require customers to authorize partners to access their data through Partner Platform. Customers can request REA to authorise new partners or add additional API access to existing partners by following the steps outlined [here](https://help.realestate.com.au/hc/en-us/articles/48886865648665-A-guide-to-data-integrations-in-Ignite-) Upon successful processing we will then add the relevant scopes to the respective partners.

Below is a summary of the current scopes available for authorised partners

-   Listing Upload API - `listing:listings:write`
-   Listing Export API - `listing:listings:export`
-   Campaign API - `campaign:listing-performance:read`,`campaign:project-performance:read`
-   Leads API - `lead:enquiries:read`
-   Webhooks API - `webhooks:subscriptions:read`, `webhooks:subscriptions:write`
