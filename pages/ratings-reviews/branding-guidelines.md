# Branding Guidelines

The ratings and reviews API allows you to display your realestate.com.au ratings and reviews on your agency website. This guide outlines the branding rules you need to follow. You can access all visual assets via [this link](https://branding.ratings-reviews.realestate.com.au/branding-assets.zip)

## Brand guidelines

### Logo

You must display the realestate.com.au logo, including ‘Ratings and Reviews provided by’ text, close to all REA rating and review content. If you have more than one REA review on a page, you only need to display the logo once on the page.

You must use file provided with the following specifications. Inverted variations will also be supplied.

![](https://partner.realestate.com.au/images/ratings-reviews/image-1.png)  

Please ensure adequate spacing from other elements on the page.

![](https://partner.realestate.com.au/images/ratings-reviews/image-2.png)  

You must not use the realestate.com.au logo in a way that implies affiliation, endorsement, or sponsorship. For example:

![](https://partner.realestate.com.au/images/ratings-reviews/image-3.png)

* * *

### Verified review emblem

Each individual review must display the verified review emblem close to the review content, using the file provided with the following specifications. An inverted variation will also be supplied.

![](https://partner.realestate.com.au/images/ratings-reviews/image-4.png)

* * *

### Star icon

You must use the star icon provided to display star ratings. This will be provided in the colours yellow (#FCCA36) and grey (#E5E3E8). Each star must follow a 1:1 ratio at a minimum of 16px by 16px.

When showing a star rating, the number must always be included to aid accessibility.

For individual ratings, the number of yellow vs grey stars must match the rating. For aggregated ratings, one star should be displayed, followed by the rating and number of reviews in brackets.

![](https://partner.realestate.com.au/images/ratings-reviews/image-5.png)

* * *

### Linking back to realestate.com.au

Each individual review must link back to the relevant agent or agency profile page. It is up to you to determine which elements are clickable, but it should be clear for the user that clicking on the link will take them to realestate.com.au. For example, you may make the verified emblem clickable, or include a text link to ‘view agent’s profile’.

The realestate.com.au logo must also always link back to the relevant agent or agency profile page, or the realestate.com.au homepage.

* * *

### Agent avatar

If you choose to display an agent avatar, we recommend a 1:1 ratio at a minimum of 24px by 24px, accompanied by the agent’s name. For example:

![](https://partner.realestate.com.au/images/ratings-reviews/image-6.png)

## Examples

### Rating-only application

Logo (inverted version) links back to realestate.com.au agency profile page.

![](https://partner.realestate.com.au/images/ratings-reviews/image-7.png)

* * *

### Rating and Review application

Each individual review links to the relevant agent’s profile page. The realestate.com.au logo links to the agency’s profile page. We recommend using a similar layout for each review, where the star rating is followed by date of review, then review content.
