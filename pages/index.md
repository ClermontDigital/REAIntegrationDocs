# REA's Partner Platform

Integrate realestate.com.au's customer data into your products with secure, production-ready APIs

## What you can build with REA APIs

Create listings

Upload and manage property listings on realestate.com.au with the Listing Upload API.

[View Listing Upload API →](./listing-upload/overview/)

Fetch leads

Receive property enquiries and leads in real-time through the Leads API.

[View Leads API →](./leads/overview/)

Display ratings & reviews

Showcase agent ratings and customer reviews on your website.

[View Ratings & Reviews API →](./ratings-reviews/overview/)

Deep link to campaign reports in Ignite

Direct users to detailed campaign performance insights in Ignite.

[View Campaign Deep Links →](./deep-links/examples/listing-performance/)

Deep link to CMA prospecting in Ignite

Enable seamless navigation to Comparative Market Analysis tools.

[View Prospecting Deep Links →](./deep-links/examples/prospecting/)

Check integration status

Monitor and verify your integration configurations.

[View Integrations API →](./integrations/overview/)
