# Changelog

## 2026-02-03

### Added

-   Support for integration events: `IntegrationCreated`, `IntegrationUpdated`, and `IntegrationDeleted` - receive notifications when integrations are created, updated, or deleted.

## 2026-01-30

### Added

-   Support for lead events: `EnquiryCreated` - receive real-time notifications when enquiries are created for listings or agencies.

## 2025-11-24

### Added

-   Public IP allow list recommendation for webhook endpoints.

### Removed

-   Requirement for new webhook subscriptions to return 401 responses to incorrect events.

## 2025-11-05

-   Changed delivery attempt retry cadence and attempt count.

## 2021-11-12

-   Added new subscription delivery endpoint that shows the detail of each attempt.

## 2020-10-21

-   Introduced Webhooks API
