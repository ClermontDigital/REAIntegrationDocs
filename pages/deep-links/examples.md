## Getting Started

Each example page includes:

-   **Web implementation** with HTML anchor tags
-   **iOS implementation** using SwiftUI Link components
-   **Android implementation** using Kotlin and Compose
-   **Best practices** and important considerations

## Important Notes

-   Always include `target="_blank"` in web links to open Ignite in a new tab
-   Replace placeholder values (like `<listingId>`, `<agencyId>`) with actual IDs when required
-   If users are not signed-in before deep linking they will be prompted to sign in first
