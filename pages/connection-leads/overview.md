# Connection Leads API

The Connection Leads API allows authenticated partners to get their connection leads against the applications approved by agents. And get the affiliated agencies for the provider.

-   [Poll connection leads](./usage.md#poll-connection-leads) - Get connection leads against the applications approved by agents.
-   [Poll affiliated agencies](./usage.md#poll-affiliated-agencies) - Get the list of agencies who have selected brands powered the authorised partner.

The Connection Leads API exposes the following APIs:

-   Retrieve all Connection Leads that are assigned to a provider
-   Retrieve all Agencies that have opted into utilising a providers services

These APIs use REA’s standard API Patterns for result paging, sorting, and filtration.

## Prerequisites

The Connection Leads API requires [authentication](../getting-started/authentication.md) via an API key. Connection Lead data is not accessible to the public and requires an approved partner agreement with REA Group.

## Rate Limits

This API follows REA’s standard [rate limiting](../getting-started/rate-limiting.md) practices.
