# Changelog

## 1.0.0 / 2021-06-06

-   Initial connection-leads-api release.
-   Poll connection leads which belong to the partner and against which applications were approved since `happenedSince`.

## 1.0.1 / 2021-06-23

-   Add `connectionProviderName` in response payload.
-   Change `agent` to `agents` in response payload.
-   Update the `street` description of `property`.

## 1.0.2 / 2021-07-30

-   Add `utilityConnectionsAllowed` in response payload.

## 1.0.3 / 2021-08-27

-   Add new endpoint to get the affiliated agencies for the provider

## 1.0.4 / 2021-09-01

-   Add `birthDate` in response payload.

## 1.0.5 / 2021-12-15

-   Add `identityDocument` in response payload.

## 1.0.6 / 2022-03-25

-   Add `passportExpiryDate` in `identityDocument` in response payload.
